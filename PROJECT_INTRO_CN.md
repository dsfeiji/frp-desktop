# FRP Desktop

FRP Desktop 是一个基于 **Go + Wails** 构建的轻量级 FRP（`frpc`）桌面客户端。  
它提供简洁的图形界面来配置服务器、选择本地端口并控制转发，无需手动编辑配置文件。

## 项目目标

- 降低 `frpc.toml` 手工配置门槛
- 提供对非命令行用户友好的操作方式
- 以统一界面支持跨平台桌面使用

## 核心功能

- 服务器设置管理（地址、端口、Token）
- 多端口一键启动/停止转发
- 运行状态与错误信息反馈
- 自动生成 `frpc.toml`
- 支持在打包产物中内置 `frpc`

## 技术栈

- 后端：Go + Wails v2
- 前端：Vite + Vanilla JavaScript + CSS
- 打包：Wails 桌面构建流程

## 使用流程

1. 打开应用并配置 FRP 服务器信息。
2. 输入一个或多个本地端口。
3. 点击“开始转发”。
4. 在界面中查看状态与日志反馈。

## 开源说明

本仓库已按公开发布场景进行整理。  
默认配置应保持占位值，真实服务器地址与凭据请在本地环境中配置，不要提交到仓库。
